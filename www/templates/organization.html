Calendário

Aprender a utlizar o calendário irá nos ajudar a organizar nosso tempo disponível além de auxiliar na programação das possíveis tarefas futuras

1. Insira todos seus compromissos fixos
Ex. (Trabalho, aula de Inglês, faculdade etc) 2. Agora preencha todas as informações adicionais
Ex. (Endereço, repetições, alarmes etc) 3. Defina uma cor para cada categoria de evento 4. Agora iremos criar essas mesmas categorias com as mesmas cores em nossa lista de tarefas (TO-DO) 5. Insira Categorias de lembretes recorrentes
Ex. (Pagamento de contas, agendar corte de cabelo, comprar ração etc) 6. Pronto, agora podemos começar a montar nossa rotina.

Definição de rotina

Agora que já temos uma noção visual de nossos compromissos, iremos demonstrar técnicas de organização para utilizarmos em nosso dia-a-dia
Tente realizar está parte sob o efeito da medicação e lembre-se que se você escolheu aquele horário sob o efeito da medicação qualquer outra razão que você tenha para não cumprila será menos inteligente, acredite.

1.Para estipularmos os horários da nossa rotina é importante saber quanto tempo levamos para cada atividade.
Marque em uma folha de papel quanto tempo cada atividade de rotina leva em média (Não se apresse para concluir a tarefa só por que o tempo está rolando, iremos ajustar os horários posteriormente) 2. Estipule um horário para dormir e para acordar, tente respeitar o período sugerido de 8h por dia 3. Configure seu alarme para sempre despertar no mesmo horário e outro com 1h de antecedencia a hora que deseja estar dormindo 4. Configure alarmes para tomar a medicação 5. É isso, agora vamos ajustar nossos ambientes.

Ambiente

O ambiente de trabalho, estudo e lazer é de fundamental influencia no desempenho de pessoas com TDAH

1. Uma hora antes de dormir tente reduzir as luzes de casa, brilho da tela do celular, e se possível utilizar luzes amarelas pois elas ajudam a induzir o corpo a relaxar. 2. Seu ambiente de estudo/trabalho deve estar limpo. 3. Caso se sinta empacado, tente mudar o ambiente que está realizando a tarefa. 4. Agora vamos ver um pouco mais detalhadamente algumas técnicas de organização.

Caixinha de entrada e saída

a caixinha de entreda e saída é essencial para minimizar os esquecimentos do dia-a-dia

1. Separe uma caixinha qualquer
Ex. (caixinha de papelão, caixinha de celular etc) 2. Posicione ela em um local que você sempre passe ao entrar ou sair de casa ou do escritório
Ex. (Próximo a porta de entrada, mesinha perto da porta do quarto etc) 3. Sempre que chegar esvazie seus bolsos e coloque tudo na caixinha
Ex. (carteira, chave do carro, lenços, moedas, celular, crachá, cartões etc) 4. Agora quarde coisas móveis como moedas, cartões etc e mantenha na caixinha somente o que for utlizar quando sair novamente 5. Criando este hábito ficará mais fácil de termos tuo que precisamos em mão todo dia

Gavetas

A organização das gavetas de casa e do escritório são ótimas para organizarmos nossos documentos

1. Crie caixinha dentro de suas gavetas, cada caixinha irá servir para uma categoria
Ex. (chaves reservas, medicamentos, produtos de uso diário, estojo com material de escritório etc) 2. Separe um espaço para colocarmos algumas pastas que ficarão de fácil acesso e outro que terá a intenção de ser um arquivo 3. Organize suas caixinhas de uma forma que você entenda ! 4. Agora vamos a organização dos documentos

Organização de documentos

Ter documentos em ordem é essencial para agiliazar cadastros e solicitações que nos são solicitados diariamente

1. Separe várias pastas plásticas e separe os documentos que você utiliza no dia-a-dia nelas
Não se esqueça de utilizar o sistema de color code para auxiliar na identificação 2. Imprima uma capa para cada pasta com uma imagem que lhe faça referencia imediata ao conteúdo nela inserido 3. Faça o mesmo procedimento para documentos que usamos com menos frequencia, podemos até utilizar pastas maiores para organiza-los
Ex. (documentos do veículo, certidões, diplomas, certificados, comprovantes etc 4. O que não for necessário jogque fora e reduzar o volume de documentos 5. Ufa... Agora vamos para a última parte onde poderemos acessar tudo de qualquer lugar

Organização Digital

Ter acesso a todos seus documentos de qualquer lugar que esteja é uma comodidade um tanto quanto indispensavel nos dias de hoje

1. Crie um serviço de armazenamento na nuvem como o Google Drive 2. Configure em todos seus dispositivos, Computadorres, celulares tablets etc 3. Organize suas pastas digitais igual as pastas física 4. Crie uma pasta de Scans, essa será uma pasta de tranferencia para enviarmos documentos rapidamente para a nuvem para se perdemos a cópia física ou se posteriormente formos organizar estes documentos em seus devidos lugares 5. Scaneie com o Scannable todos seus documentos 6. O que não for necessário manter uma cópia em papel, jogue fora ! 7. Configure as tags ou cores de suas pastas digitais igual as pastas físicas 8. Após esse extensivo processo terminamos as dicas de utilização
